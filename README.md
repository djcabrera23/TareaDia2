# Desarrollo de software #

### El desarrollo de software es el conjunto de actividades informáticas (creación, diseño, implementación y soporte) destinadas a construir los programas e instrucciones que permiten a las computadoras ser programables y funcionales, independientemente del hardware.
###

### El desarrollo de software se define como un proceso integral que fusiona la creatividad con la técnica para transformar ideas en soluciones digitales funcionales. No se limita únicamente a la escritura de código, sino que abarca un ciclo de vida completo: planificación, diseño, programación, pruebas y soporte.
###

### Tipos de software ###

- **Software de Sistemas:** Es la base que permite el funcionamiento del hardware. Incluye sistemas operativos, gestores de discos y herramientas de mantenimiento que coordinan los recursos del ordenador.

- **Software de Programación:** Son las herramientas técnicas utilizadas por los desarrolladores para crear nuevos programas. Ejemplos clave son los editores de texto, compiladores, depuradores y enlazadores de código.

- **Software de Aplicación (*Apps*):** Programas diseñados para que el usuario final realice tareas específicas. Abarca desde suites de oficina y reproductores multimedia hasta aplicaciones móviles y plataformas de redes sociales.

- **Software Integrado (*Embebido*):** Diseñado para controlar dispositivos que no son computadoras convencionales, como sistemas automotrices, robótica industrial y redes de telecomunicaciones. Estos sistemas son la base del Internet de las cosas (IoT).
